# Biztrack

Biztrack is a full-stack business tracking application with a React/Vite frontend and an Express/Prisma backend.

## Getting Started

Install frontend dependencies:

```bash
npm install
```

Install backend dependencies:

```bash
cd backend
npm install
```

Copy the example environment files and fill in local values:

```bash
cp .env.example .env
cp backend/.env.example backend/.env
```

Run the frontend:

```bash
npm run dev
```

Run the backend:

```bash
cd backend
npm run dev
```
